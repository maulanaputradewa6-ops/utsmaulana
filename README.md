# Tugas Portofolio Web - Maulana

Repository ini berisi tugas akhir mata kuliah Pemrograman Web. Website portofolio ini dibangun menggunakan Next.js dan Tailwind CSS dengan data statis (tanpa database).

# Teknologi
- Next.js 14 (App Router)
- Tailwind CSS
- TypeScript

# Fitur Utama
- Layout sudah responsif untuk HP, tablet, dan desktop.
- Tombol dark mode / light mode.
- Halaman detail portofolio menggunakan dynamic route (`[id]`).
- Komponen interaktif (form kontak dan tombol tema) menggunakan `useState`.

# Cara Menjalankan di Lokal
Untuk menjalankan project ini di komputer, ikuti langkah berikut di terminal:

1. Clone repository:
   ```bash
   git clone [https://github.com/maulana/UTS-web-maulana.git](https://github.com/dendra14/UTS-web-maulana.git)

2. Masuk ke folder:
    cd UTS-web-maulana

3. Install package:
   npm install

4.  Jalankan server:
   npm  run dev

5. Buka http://localhost:3000 di browser.
